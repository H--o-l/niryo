niryo-http
==========


Info
----

Basic Niryo HTTP API, on top of official Python 2.7 API.

To be deployed on niryo raspberry, and exec as "niryo" user.


Using (dev)
-----------

.. code:: shell

 python setup.py sdist



Installing
----------

.. code:: shell

 python setup.py install



Packaging
---------

.. code:: shell

 python setup.py sdist
